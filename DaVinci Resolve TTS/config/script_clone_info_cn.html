MiniMax 克隆说明

☑ 已有克隆音色

  • 如果已经从MiniMax官方克隆了音色，直接填入 音色ID 和 音色名称，点击添加后， 音色列表中即可看到您的克隆音色！

☐ 已有克隆音色
（从达芬奇导出音频进行克隆）

    • 步骤一：确保时间线上已有要克隆的音频。
    • 步骤二：按 M 键 添加标记点， 按住 Alt 键 拖拽选取音频区间（10 秒–5 分钟）。
    • 步骤三：填写 音色名称 和 音色ID， 根据需要勾选 降噪 和 音频统一 选项。
    • 步骤四：输入 试听文本， 克隆完成后试听音频会自动加载到时间线进行预览。
    • 步骤五：点击克隆，克隆完成后 克隆ID 将出现在音色列表中。

    费用说明

    • 确保 minimax 账户余额充足，9.9 元/音色。
    • 克隆音色时不会立即扣费，费用将在首次使用该音色合成语音时收取。
